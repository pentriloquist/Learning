How to Make Tables with HTML

  1. First use the opening and closing <table> tags.
  2. To make the table headers, nest the opening and closing <thead> tags within the <table> tags.
  3. Follow your <thead> with as many <tbody> tag. Don't forget to close it.
  4. Nested within your body, fill it with rows with as many <tr> tags as you so desire. Close them!
  5. To fill these rows with information, use the <td> opening and closing tags.
  6. You can span data across multiple columns and/or rows using the <colspan>/<rowspan> tags. Include a ="#" after the tag to specify how many columns/rows the data shall span (obviously the # should be replaced with the appropriate number)
